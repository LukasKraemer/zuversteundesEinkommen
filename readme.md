Ich bin verantwortlich für die Einhaltung des Recht am eigenen Bild, alle anderen gesetzlichen BEstimmungen werden von unserer Berufsschullehrerin Sima Darabi übernommen.

- HTML Programmerung
- einfache CSS formatierungen ohne Responsive Ansicht
- die Benutzung von einigen JQuery Funktionen
- Dem Pflichtenheft mit 25%
